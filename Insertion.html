¿Cómo funciona?

Insertion Sort construye el array ordenado un elemento a la vez. Toma cada elemento y lo inserta en su posición correcta dentro de la parte ya ordenada, desplazando los elementos mayores hacia la derecha.

Complejidad
  • Mejor caso: O(n) - Array ordenado
  • Peor caso: O(n²) - Array invertido
  • Espacio: O(1) - In-place
  • Estable: Sí
Ventajas
  • Eficiente para arrays pequeños
  • Funciona bien con datos casi ordenados
  • Algoritmo estable
  • Ordenamiento en línea (online)
