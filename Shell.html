Shell Sort: Una generalización del Insertion Sort que permite el intercambio de elementos que están lejos entre sí. Utiliza una secuencia de gaps para ordenar subarrays.

Mejor Caso
O(n log n)
Caso Promedio
O(n log² n)
Peor Caso
O(n²)
// Secuencia de gaps (Shell original)
for (let gap = Math.floor(n/2); gap > 0; gap = Math.floor(gap/2)) {
  for (let i = gap; i < n; i++) {
    let temp = array[i];
    let j;
    for (j = i; j >= gap && array[j - gap] > temp; j -= gap) {
      array[j] = array[j - gap];
    }
    array[j] = temp;
  }
}
