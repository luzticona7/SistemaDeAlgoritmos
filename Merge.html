Merge Sort: Es un algoritmo de ordenamiento eficiente que utiliza el enfoque "divide y vencerás". Divide el array en mitades, ordena cada mitad recursivamente y luego las combina.

Mejor Caso
O(n log n)
Caso Promedio

Peor Caso
O(n log n)
// Secuencia de gaps (Shell original)
for (let gap = Math.floor(n/2); gap > 0; gap = Math.floor(gap/2)) {
  for (let i = gap; i < n; i++) {
    let temp = array[i];
    let j;
    for (j = i; j >= gap && array[j - gap] > temp; j -= gap) {
      array[j] = array[j - gap];
    }
    array[j] = temp;
  }
}
